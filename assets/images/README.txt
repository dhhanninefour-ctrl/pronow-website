===============================================================
PRONOW 웹사이트 — 이미지 넣는 방법 (How to add images)
===============================================================

이 폴더(assets/images/)에 아래 "파일 이름"으로 사진을 저장하면,
새로고침했을 때 사이트의 해당 위치에 자동으로 표시됩니다.

- 지원 형식: .webp, .jpg, .jpeg, .png  (아무거나 하나면 됩니다)
- 가로세로 비율은 안내된 모양에 맞추면 가장 예쁩니다.
- 사진이 없으면 회색 박스(PHOTO/IMAGE)가 그대로 보입니다 — 정상입니다.

회사소개서 PDF에서 캡처/내보내기 한 사진을 아래 이름으로 저장하세요.

---------------------------------------------------------------
[ 메인 상단 배경 ] — 가로로 넓은 사진 권장 (코트 전경 등)
  hero-court.jpg
  (회사소개서 7페이지의 코트 사진 추천)

팁: 사진 파일 용량이 너무 크면(예: 5MB 이상) 로딩이 느려집니다.
    가능하면 가로 1600px 내외로 줄여서 저장하세요.
